Java requirements for different Minecraft Versions

Depending on the version of Minecraft you are looking to host for, there are different Java requirements.

Minecraft Java Requirements (Release Versions)

Game Version Minimum Java Version
1.21 - 1.20.5 Java SE 21
1.20 - 1.18 Java SE 17
1.16 - 1.12 Java SE 8
1.11 - 1.6.1 Java SE 6
1.5 - 1.0 Java SE 5
Tip: Older versions of Minecraft can run on newer versions of Java, so for security purposes, it is best to use the latest LTS (Long Term Support) purposes.
