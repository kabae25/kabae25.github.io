Minecraft Server Software

A list of different Minecraft game server software

Vanilla

Servers that do not support mods or plugins.

Server Software Description Link
Vanilla Official server developed and maintained by Mojang Studios https://www.minecraft.net/download/server
MCHPRS A multithreaded Minecraft server built for redstone. https://github.com/MCHPR/MCHPRS

Plugins

Servers that support plugins

Server Software Description Link
Bukkit/CraftBukkit Server that supports Bukkit Plugins https://www.spigotmc.org/wiki/buildtools/
Spigot Fork of Bukkit with performance enhancements and additional features (Bungeecord Support) https://www.spigotmc.org/wiki/buildtools/
Paper Fork of Spigot with performance enhancements and glitch patches. https://papermc.io/
Pufferfish A fork of Paper designed for large servers https://github.com/pufferfish-gg/Pufferfish
Purpur A fork of Pufferfish adding "fun" configurations https://github.com/pl3xgaming/Purpur

Mods

Servers that support client side mods

Server Software Description Link
Forge Server that allows for Forge Mods http://files.minecraftforge.net/
Neoforge A fork of Forge by other developers https://neoforged.net/
Fabric Server that allows for Fabric Mods https://fabricmc.net/
QuiltMC A fork of Fabric that allows for Quilt/Fabric Mods https://quiltmc.org/

Proxies

Proxies that connect servers together

Server Software Description Link
Bungeecord A proxy that links Spigot servers together. https://github.com/SpigotMC/BungeeCord
Velocity A proxy that focuses on support, scalability, and flexibility https://www.velocitypowered.com/
