Using commands

Allow a player to enter commands by granting them Operator status..

  1. From the server console, enter the command op <player>, repacing <player> with your in-game username. The server will display an output message.
    Note: Entering commands in the server console does not require the / prefix.
  2. In-game, enter the command /tp ~ ~ ~ and press enter.
    The server will return a message beginning with "Teleported <player> to..."
    Note: To revoke Operator status, use the /deop <player> command.
