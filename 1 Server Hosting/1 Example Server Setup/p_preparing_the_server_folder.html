Preparing the Server folder

Make a new folder on your desktop for the server. The name of the folder does not influence the server properties, and players will not be able to see it.

  1. On the desktop, right click to bring up the context menu. Hover over New then click Folder from the list of available options.
  2. When prompted for the name of the folder, enter My Minecraft Server and click away to name the folder.
  3. Navigate to the Downloads folder and move the jar file downloaded in Downloading a Minecraft Game Server.

    Result: The server folder setup is complete. Continue to Initializing the Server
