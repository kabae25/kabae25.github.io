Setting a Server Icon

A server icon is a great way to personalize your server.

  1. Open the image you want to use as a server icon with the Photos app.
  2. Right Click over the image and select Resize image
  3. Set the Width (px) and Height (px) both to 64, and the File type to PNG
    Note: If necessary, click the link icon between the Width and Height fields to break the aspect ratio.
  4. Click Save
  5. Save the image as server-icon to the main folder of your server.
  6. Reload the server with /reload followed by /reload confirm
  7. Refresh the connections of the server list in-game
