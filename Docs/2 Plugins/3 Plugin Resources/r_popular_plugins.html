Popular Plugins

A list of popular plugins.

EssentialsX

EssentialsX is a large quality of life mod that adds a variety of useful plugins for Players and server administrators.

https://essentialsx.net/

WorldEdit

An in-game Map editor that has support for selections, schematics, copy/paste, brushes and scirpting.

https://dev.bukkit.org/projects/worldedit/

Dynmap

Host a webserver that displays the in-game map.

https://www.spigotmc.org/resources/dynmap%C2%AE.274/

mcMMO

A plugin that adds a leveling system in-game.

https://www.spigotmc.org/resources/official-mcmmo-original-author-returns.64348/

Multiverse

A plugin that allows servers to run multiple worlds and teleport between.https://hangar.papermc.io/Multiverse/Multiverse-Core

LuckPerms

A feature rich Permission Manager with large community support.

https://luckperms.net/
