Organizing Plugins

Example best practices with multiple plugins.

Renaming Plugins

Because Plugin jar files cannot be within subfolders in the plugins folder, organizing plugins by adding a prefix to the file can help in sorting by purpose or feature.

For example, adding c_ before a plugins that relate to chat. Or a_ for admin plugins.

Note: Renaming plugins does not affect usage or performance.

Inactive Folder

Depending on the number of plugins, it can be useful to make a Inactive Plugins folder inside the server's plugins folder to hold unused jar and configuration files.
