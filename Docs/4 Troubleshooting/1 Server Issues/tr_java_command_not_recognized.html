Java command not recognized

Condition

Cause

java command is not recognized because Java has not been fully installed.

Remedy

  1. Restart Your Computer
  2. Open the Command prompt by pressing Win + R and entering cmd
  3. Type java -version
    Note:

    If java is still not recgonized, re-install Java.
