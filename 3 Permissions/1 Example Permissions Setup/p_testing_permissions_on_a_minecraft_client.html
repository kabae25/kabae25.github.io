Testing Permissions on a Minecraft Client

Testing Permissions in-game with a Minecraft Client

  1. Remove Operator status from the Player. Run the /deop <Player> command.
  2. Use a command that the player has permissions to. Run the /kick <Player> command on yourself.
The Player has demonstrated the ability to use a permission inherited by the Admin group. Congratulations on taking the first steps into large-scale server administration!
